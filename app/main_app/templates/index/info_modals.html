About Default Schedule

The default schedule is good when you have regular schedules like: 6a-2p. They will appear as the help dropdown when you will put into schedules for employess. Time have to be in proper format, for example: "11a-9p". To edit default schedule click on default schedule name.

About Departments

Departments is for you to have track on your employees, in each department. If your company don't have any departments, add at least one and when you will be creating employess add all of them to your default department. To edit department click on deaprtment name.

About Employees

Create employees in order to create schedule. Unique employee number presented as the integer number. If your company doesn't have unqiue employee numbers, add some random numbers for each employee, unique for each. To edit employee click on employee name.

About Schedules

Create new schedule for every employee by using option "+ Add new row". If you have default schedules - they will appear as the options in Day x time. Adding duty is optional. For day off leave both day and duty fields empty. To edit employee's schedule click on employee's name. While changing the schedule, you can change employee, that attached to current schedule, as soon as new employee aren't into the schedule yet. Push "Export" option to export schedule table as .csv file.
